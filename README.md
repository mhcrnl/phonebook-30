# Phonebook-30
Agenda telefonica implementata in limbajul de programare C.

## Instalare
```
git clone https://github.com/mhcrnl/phonebook-30.git
cd phonebook-30
gcc phonebook.c
```

## Utilizare
```
./phonebook
```

## API
```
void menu();
void chooseOption(char OPTION);

void displayAllContacts();

void displayByFirstName();
```
